## [0.1.0] - 2026-04-16

### Added

- `pixsize scan` -- scan directories for image metadata with filters
- `pixsize check` -- validate images against YAML rules or CLI flags
- `pixsize resize` -- batch resize with 14 presets or custom dimensions
- `pixsize rename` -- rename by pattern tokens (dimensions, date, index)
- JSON, CSV, and table output formats
- Recursive and non-recursive directory scanning
- Aspect ratio preservation in resize
- Dry-run mode for rename preview
